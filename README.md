# Sambr News Bot

Бот для моніторингу новин Самбірщини та повітряних тривог.

## Функції
- Моніторинг повітряних тривог через API alerts.in.ua
- Автоматична публікація новин про світло/графіки
- Меню для користувачів з можливістю поділитися інформацією
- Адмін-панель для модерації
